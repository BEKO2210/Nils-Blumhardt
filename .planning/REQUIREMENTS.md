# Requirements: Blumhardt Maler Website

## v1 Requirements

### Structure (STRUCT)

- **STRUCT-01**: One-Pager mit Scroll-Sektionen (Hero, Leistungen, Galerie, Bewertungen, Kontakt)
- **STRUCT-02**: Mobile-First responsive Design mit blauer Akzentfarbe
- **STRUCT-03**: Integration des bestehenden Firmenlogos

### Features (FEAT)

- **FEAT-01**: Anfrage-Formular via Formspree mit E-Mail-Weiterleitung an N.Blumhardt@t-online.de
- **FEAT-02**: WhatsApp-Button für Direktkontakt (0160 5972245)
- **FEAT-03**: Statisches Google-Bewertungen Widget (5.0 Sterne, 4 Bewertungen)
- **FEAT-04**: Projekt-Galerie für 10-25 Referenzfotos mit Lightbox

### Compliance (COMP)

- **COMP-01**: Impressum (DSGVO-konform)
- **COMP-02**: Datenschutzerklärung (DSGVO-konform)
- **COMP-03**: SEO-Optimierung für lokale Suche (Freiberg, Ludwigsburg, Stuttgart)
- **COMP-04**: Schnelle Ladezeiten (<3s auf Mobile, Lighthouse >90)

## v2 (Future)

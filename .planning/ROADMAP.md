# Roadmap: Blumhardt Maler Website

## Overview

Mobile-First One-Pager für einen Malerbetrieb: Von der Grundstruktur über interaktive Features bis zur fertigen, SEO-optimierten Website.

## Phases

- [x] **Phase 1: Foundation & Layout** - HTML-Struktur und CSS-Designsystem
- [x] **Phase 2: Features & Content** - Interaktive Elemente und Inhalte
- [ ] **Phase 3: Polish & Compliance** - SEO, Rechtliches, Performance

## Phase Details

### Phase 1: Foundation & Layout
**Goal**: Vollständige HTML-Struktur mit allen Sektionen und Mobile-First CSS-Designsystem
**Depends on**: Nothing (first phase)
**Requirements**: STRUCT-01, STRUCT-02, STRUCT-03
**Success Criteria** (what must be TRUE):
  1. Website lädt und zeigt alle Sektionen (Hero, Leistungen, Galerie, Bewertungen, Kontakt)
  2. Layout ist responsive und sieht auf Mobile gut aus
  3. Blaue Akzentfarbe ist durchgängig implementiert
  4. Logo ist sichtbar im Header
**Research**: Unlikely (standard HTML/CSS patterns)
**Plans**: TBD

Plans:
- [x] 01-01: HTML structure and CSS design system

### Phase 2: Features & Content
**Goal**: Alle interaktiven Features funktionieren (Formular, Galerie, WhatsApp, Reviews)
**Depends on**: Phase 1
**Requirements**: FEAT-01, FEAT-02, FEAT-03, FEAT-04
**Success Criteria** (what must be TRUE):
  1. Kontaktformular sendet E-Mails via Formspree
  2. WhatsApp-Button öffnet Chat mit korrekter Nummer
  3. Galerie zeigt Bilder und öffnet Lightbox bei Klick
  4. Google-Bewertungen Widget zeigt 5.0 Sterne
**Research**: Unlikely (Formspree docs, standard lightbox)
**Plans**: TBD

Plans:
- [x] 02-01: Contact form with Formspree
- [x] 02-02: Gallery with lightbox and interactive elements

### Phase 3: Polish & Compliance
**Goal**: Website ist SEO-optimiert, rechtlich konform und performant
**Depends on**: Phase 2
**Requirements**: COMP-01, COMP-02, COMP-03, COMP-04
**Success Criteria** (what must be TRUE):
  1. Impressum und Datenschutz sind vollständig und verlinkt
  2. Meta-Tags für lokale Suche sind gesetzt
  3. Lighthouse Mobile Score >90
  4. Ladezeit <3s auf Mobile
**Research**: Unlikely (standard SEO/legal patterns)
**Plans**: TBD
